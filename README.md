# Solavisetech-women-in-data-cohort2
# About the Program: 
The solavise Tech women in Data program is a 2 Months intensive learning initiative, organised by Solavise Technology Institute, to empower women with foundational and practical data skills for the modern digital workplace.The programm also aims at introducing women to essential data and collaboration tools such as python, Github,LinkedIn, Rezi AI, data analysis and data visualisation. Through live interactive sessions, hands-on projects, peer learning pods and mentorship support, participants gain practical experience while building confidence in the tech space. This program also promotes collaboration among women in technology, encourages professional networking, and prepares participants for remote work and global digital opportunities. The learning journey concludes with a creative hackathon experience where participants apply their skills to real-world projects, recieve certificates and awards for outstanding performance.
